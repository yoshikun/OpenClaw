## 3. 技能效果（核心机制）

> **技能机制**：每使用 1 张咒术或反制，就召唤 1 个 1/1 的火精灵（每回合一次）

## 5. 链式礼包功能（新系统）

### 5.1 礼包状态规则

| 状态 | 条件 |
|------|------|
| 购买状态 / 可领取状态 | 玩家可进行购买/领取操作，完成操作获得礼包内道具 |
| 售罄状态 | 玩家无法进行操作 |
| 已解锁状态 | 前置礼包已售罄，玩家可点击购买按钮 |
| 未解锁状态 | 前置礼包未售罄，点击购买按钮无效（锁图标） |

### 5.2 基本规则

1. **首位礼包**：默认「购买状态 / 可领取状态」
2. **后续礼包**：需前置礼包为「售罄状态」才变为「已解锁状态」，否则为「未解锁状态」
3. **购买方式**：糖果购买 / 免费领取
